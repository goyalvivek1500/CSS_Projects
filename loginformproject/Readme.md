# Login Form (CSS Only)

## 📌 Overview

This is a simple login form UI designed using CSS. It focuses on clean layout, centered positioning, and basic styling for form elements like input fields and buttons.

---

## 🎨 Features

* Centered layout using Flexbox
* Clean and minimal design
* Styled input fields and button
* Rounded corners for better UI
* Proper spacing and alignment

---

## 🛠️ Styling Details

### Body

* Uses `display: flex` to center the form
* `justify-content: center` and `align-items: center` for alignment
* `height: 100vh` for full screen
* Light background color

### Container (.box)

* Fixed width (400px)
* Padding for spacing
* White background
* Rounded corners using `border-radius`

### Heading (h1)

* Center aligned
* Margin at bottom for spacing

### Labels

* Medium font size
* Slightly bold for readability

### Input Fields

* Full width
* Padding for better user experience
* Rounded corners
* Margin at bottom for spacing

### Button

* Full width
* Larger font size
* Rounded edges
* Top margin for spacing

---


## 🚀 Usage

Link this CSS file to your HTML login form and it will automatically apply styling.
